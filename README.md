### 『仿知乎专栏』本项目是高仿知乎专栏的前端项目，使用 Typescript + Vue3.0 从零到一到上线 高仿知乎专栏

### 安装依赖

```
npm install / yarn
```

### 运行本地开发环境

```
npm run serve
```
